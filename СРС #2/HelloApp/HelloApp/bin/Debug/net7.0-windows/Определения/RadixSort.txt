Радиксная сортировка

Радиксная сортировка — это алгоритм линейной сортировки, который сортирует элементы, обрабатывая их поразрядно. Это эффективный алгоритм сортировки целых чисел или строк с ключами фиксированного размера.

Вместо прямого сравнения элементов радиксная сортировка распределяет элементы по сегментам на основе значения каждой цифры. Повторно сортируя элементы по их значащим цифрам, от наименее значимого к наиболее значимому, Radix Sort достигает окончательного порядка сортировки.


Шаги процесса сортировки:

Шаг №1:
Находим в массиве максимальное число.

Шаг №2:
Надо узнать сколько разрядов у максимального числа.

Шаг №3:
По разраядам элементов будем считывать. Сперва единицы, десятки (если есть) и так далее. У нас есть массив arrayCount[10] и там будет идти процесс считывания. Каждый индекс массива является уникальной цифрой.

Шаг №4:
Затем элемениы в массиве arrayCount суммируется. Это для того, чтобы знать индексы для сортировки в массив arrayOutput.

Шаг №5:
Начинаем вставку чисел из массива array в массив arrayOutput с помощью массива arrayCount. Прогон массива array происходит с поледнего элемнта, то есть справа налево.

Шаг №6:
Массив arrayOutput сохраняется в массив array.

Процесс сортировки будет продолжаться до последнего разряда максимального числа в массиве array.
